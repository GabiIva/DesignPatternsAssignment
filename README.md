# Проект по Шаблони за софтуерен дизайн

Това хранилище съдържа примери за два шаблона за създаване на обекти:

1. **Simple Factory (на C#):** Капсулиране на логиката за инстанциране в един клас.
2. **Factory Method (на Java):** Използване на абстрактен метод за делегиране на създаването към подкласове.
