# Проект по Шаблони за софтуерен дизайн

Това хранилище съдържа собствени реализации на два шаблона за създаване:

1. **Simple Factory (C#):** Капсулиране на логиката за инстанциране в един клас чрез статичен метод.
2. **Factory Method (Java):** Използване на абстрактен метод за делегиране на създаването към конкретни подкласове.
